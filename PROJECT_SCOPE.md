# Product

A web app that generates a production-ready Next.js starter as a ZIP download.

# Target user

Solo developers who want auth + database setup done quickly.

# v1.0.0 includes

- Web app
- Next.js only
- MongoDB only
- Better Auth only
- ZIP download
- .env.example
- README
- protected dashboard example

# v1.0.0 does NOT include

- GitHub repo creation
- Stripe
- Email
- RBAC
- multi-tenancy
- multiple frameworks
- CLI

# Success criteria

A user can:

1. choose options in wizard
2. download ZIP
3. run `npm install`
4. run `npm run dev`
5. get a working app without fixing generator errors
